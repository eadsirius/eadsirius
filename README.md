## 라이브러리

- react-native-store


  스트링기반 데이터 쉽게 입출력해주는 라이브러리

  > 로그인시 아이디, 토큰 저장(자동로그인)을 위하여 설치하였다.

- react-native-stack
  화면 스크린 나누워 페이지 전환을 위해 설치

- react-native-safe-area-context
  아이폰떄무네 나왔다고 한다
